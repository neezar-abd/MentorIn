# Satoshi Font

Font Satoshi by Indian Type Foundry

## Installation Instructions:

1. Download Satoshi font from: https://www.fontshare.com/fonts/satoshi
2. Extract the font files
3. Copy the following files to this directory:
   - Satoshi-Variable.woff2
   - Satoshi-Variable.woff
   - Satoshi-VariableItalic.woff2
   - Satoshi-VariableItalic.woff

Or download directly from CDN (already configured in layout.tsx)
